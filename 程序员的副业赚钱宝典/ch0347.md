# 程序员的副业：基于里程积分套利脚本与管家服务的“常旅客”中台

传统的常旅客社交是“薅羊毛群”，架构师的方案是建立一套**自动化里程兑换预言机**。利用 Python 爬虫解析全球航空/酒店联盟的可用仓位（Availability）与汇率差（Spread），通过“权益核销”将冗余积分转化为高单价的头等舱资源流，构建商旅市场的“套利银行”。

## 01. 需求验证与 MVP 策略
*   **痛点**：核心痛点是“积分难用”和“动态贬值”。中产用户持有大量信用卡/飞行积分，却不懂如何寻找跨航司的最优兑换路径（如：用维珍里程换全日空头等舱）。
*   **MVP 方案**：
    *   **不要做**：开发一个类似 ExpertFlyer 的全自动实时订票系统。
    *   **要做**：建立一个 Python 脚本，每日监控 20 条热门回国/度假航线的“里程票位”。由于你具备数据处理能力，将这些信息整理成“今日里程红利表”，通过私域会员制发布。

## 02. 本土化流量实操
*   **公域截流**：
    *   **小红书**：发布“500 块玩转公务舱：全网最全里程兑换公式”图文。
    *   **飞客茶馆/知乎**：撰写深度技术贴，分析特定航司（如国泰、南航）的票池算法及放票规律，建立专业权威感。
    *   **视频号**：拍摄“积分实操”短视频，展示你如何通过代码脚本秒杀掉一张热门日期的奖励票。
*   **私域转化**：
    *   提供“1对1 里程资产规划”，帮用户梳理名下所有积分，输出最优利用路径。

## 03. 💰 生意账本
### 💰 财务测算（预估模型）
*   **启动成本**：¥300（云服务器 + 抢票 API 订阅）
*   **单客模型**：
    *   客单价：¥199（入群费/月）或 ¥2,000+（单次代换服务费）
    *   获客成本 (CAC)：约 ¥80（精准社群广告/内容导流）
    *   交付成本：10 分钟脚本查询 + 30 分钟下单操作
    *   **单客毛利**：¥1,500+ (毛利率 80% 以上)
*   **回本周期**：由于是服务型，首单即回本。
*   **收入天花板**：个人操作上限约 ¥2.5万/月；发展下级二级代理后可达 ¥20万/月。

## 04. 📊 核心指标仪表盘
*   **北极星指标**：单客积分兑换价值增长率（Point Value Uplift）。
*   **KPI**：脚本查询成功率；代换任务准时交付率。
*   **OKR**：季度内打通 5 个主流航司联盟的底层 API 爬取。

## 05. ⚠️ 避坑指南
*   **法务风险**：严禁买卖里程积分，必须以“代订/管家服务”名义操作。
*   **平台风险**：频繁爬取会触发反爬，需搭建高质量代理 IP 池及 Headless UI。
*   **确认风险**：里程票确认周期较长，且不支持退换，必须与客户签署免责声明。

## 06. 案例模拟
程序员**小B**以前在各大论坛义务解答里程问题，后来发现自己掌握的查询逻辑可以产品化。他写了一个自动化监控脚本，专门盯“中美往返”的极稀缺票位。
第一周，他在飞客发了 3 篇干货贴，引流了 50 人进微信。
第二周，他帮一位着急回国的留学生用 5 万里程抢到了一张价值 ¥2.5万 的商务舱，收取了 ¥2,000 的技术服务费。
困难是：航司反爬升级。解决办法是：放弃高频爬取，改为订阅海外商业数据 API（如 AwardHacker）并进行二次逻辑清洗。
