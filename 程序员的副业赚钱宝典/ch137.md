# 程序员的“英语出海”：面向开发者的技术英语私教 (ch137.md)

对于程序员，教英语不是教语法，而是教“如何流畅阅读英文文档”和“如何参与 GitHub 社区协作”。

---

## 01. 需求验证与MVP策略

*   **痛点**：国内开发者想看外刊、想投外企、想写高质量的英文注释，但在传统的四六级语境里，完全不懂技术专有名词。
*   **MVP方案**：**严禁自研 App！**
    1.  第一步：在微信群做 7 天“英文文档朗读营”。
    2.  第二步：每天选一篇顶级开源项目的 Readme 或外文博客，做带注释的翻译。
    3.  第三步：验证学员愿意为“纠错”和“翻译精准度”付费的意向。
    4.  成本：¥0。

---

## 02. 本土化流量实操

*   **引流渠道**：
    *   **公域截流**：B站/小红书发布“我是如何从 0 基础到能跟印度老外吵架的”经验帖。
    *   **知乎/掘金**：针对“技术英语怎么学”提供系统路线图。
*   **转化钩子**：加群领《2025 年必读的 100 个技术核心单词表》。

---

## 03. 💰 生意账本（月度模型）

以一个小班（20人班）为例：

*   **启动成本**：¥0。
*   **单客模型**：
    *   **训练营价格**：¥499 - ¥999 / 期（4周）。
    *   **交付人力**：每天 1 小时批改。
    *   **获客费**：¥50 / 人。
    *   **单客毛利**：¥400 - ¥900 (毛利率 > 90% )。
*   **收益分析**：月入 ¥1万 - ¥2万（轻量化操作）。
*   **回本周期**：首个学员即回本。

---

## 04. 📊 核心指标仪表盘 (Dashboards)

*   **北极星指标**：**学员自主贡献（PR/Issue）成功率**。真正能产出才是硬指标。
*   **KPI**：
    *   **单词卡完成率**：每日打卡率需 > 80%。
    *   **口语沟通自信度自测表**（开营收盘对比）。
*   **OKR**：
    *   [O] 帮助 100 个开发者通过国际化远程办公面试。
    *   [KR] 与 3 家猎头公司达成“英文面培”战略合作。

---

## 05. ⚠️ 风控与避坑

*   **法务风险**：严禁大规模复印未经授权的外文教科书。**方案**：直接使用开源文档（MIT/Apache 协议）作为教材。
*   **交付风险**：英语学习极易中途放弃。**方案**：引入“奖金池”制度，全勤打卡返现。

---

## 06. 模拟实战案例

**小A（某海外转国内的架构师）**：
他没教 ABC，而是专门教“如何写出符合大厂标准的 PR 说明”。他把学员在 GitHub 的改动做成合集，在推特上宣传。结果不仅吸引了学生，还吸引了硅谷的创业公司找他做内推。第一期 20 人满员，不仅赚了学费，还赚了 3 笔内推成功的“赏金” ¥3万。
