# 程序员的“降维导学”：基于 AI 的辅助学习系统与定制化提分路径 (ch160.md)

对于程序员，教孩子写作业不是卖体力陪读，而是卖“知识图谱的可视化”和“基于 AI 的自适应刷题流”。

---

## 01. 需求验证与MVP策略

*   **痛点**：家长辅导作业鸡飞狗跳，孩子对抽象概念（如：物理力学、数学函数）理解困难。传统的培训班只有“大锅饭”，缺乏针对个人薄弱点的精准打击。
*   **MVP方案**：**严禁自己研发教学平台！**
    1.  第一步：先为自己的孩子或亲戚的孩子，利用 ChatGPT/Claude 配合特定的 Prompt 建立一套“错题举一反三”系统。
    2.  第二步：录制一段“如何用 5 分钟讲透一个复杂物理公式”的可视化演示视频。
    3.  第三步：在家长群发布“AI 辅助高效导学”实验班，收费 ¥199。
    4.  成本：¥0。

## 03. 💰 生意账本（单人模型）

以一次“深度提分方案定制 + 4 周 AI 陪练监控”为例：

*   **启动成本**：¥0。
*   **单客模型**：
    *   **方案定制费**：¥500 - ¥1,500 / 次。
    *   **月度系统维护/答疑费**：¥200 - ¥500 / 月。
    *   **单客总收益**：¥700 - ¥2,000。
    *   **交付人力**：前期配置 2 小时，后续每周 15 分钟监控。
*   **收益分析**：月服务 30 人，月入 ¥2万 - ¥4万，且随着 Prompt 库的积累，人效比极速提升。
*   **利润率**：100%（不计入自有时间）。

## 05. ⚠️ 风控与避坑

*   **合规风险**：严禁代写作业。**方案**：系统的核心是“思路启发”和“知识点补全”，而不是直接给答案。

## 06. 模拟实战案例

**小A（某资深 Python 讲师）**：
他没去培训班代课，而是专门为“偏科生”定制 AI 陪教。他利用 API 搭建了一个简单的对话框，后端接入了所有主流考点。当学生做不出题时，AI 不是给答案，而是给 3 个提示。第一月他招了 10 个学生，平均每人收费 ¥1500。因为孩子不仅不反感，反而觉得像玩游戏一样，家长评价极高。他现在不仅成了本地的“升学顾问”，还收到了风投的早期关注。
